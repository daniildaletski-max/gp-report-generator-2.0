# Google Sheets Error File Structure

## File: Copy of 12. December Errors File .xlsx
Sheet: "Error Count"

## Columns:
- Column A: Empl_ID (empty)
- Column B: Employee Name (GP name)
- Column C: Work load
- Column D: Total Errors (error count)
- Column E: Errors per FTE

## Data starts from row 2 (row 1 is header)
